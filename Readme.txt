The Resampler's solution developed with VS 2012 contains 3 projects:
1) resampler : Core resampler source code based on c that generate resampler.dll.

2) reSampleDemo : Is c# Console Application while demonstrate and test how to invoke the function and param on resampler.dll

3) thinkgear_testapp : Its C/C++ Application to test how to call resampler.dll on C/C++.
